# Hasaan Hameed - Portfolio

A modern, responsive portfolio website showcasing professional experience, technical skills, and selected projects.

## Live Link

[hasaanhameed.vercel.app](https://hasaanhameed.vercel.app)

## Overview

This is a personal portfolio website built with React and TypeScript, featuring a clean, professional design with smooth navigation and animated elements. The site showcases work experience, technical expertise, and completed projects with links to live demos and source code.

## Features

- Responsive design optimized for mobile and desktop
- Smooth section navigation with scroll animations
- Project showcase with GitHub and demo links
- Experience timeline with detailed project descriptions
- Skills organized by category
- Contact information with email and phone options
- Lottie animation background in hero section
- Dark/light theme compatible styling

## Tech Stack

- React 18
- TypeScript
- Vite
- TailwindCSS
- Lucide React Icons
- Lottie React
- ESLint

## Sections

**About**
Brief introduction and contact information with direct links to GitHub, LinkedIn, email, and phone.

**Experience**
Detailed timeline of professional roles including achievements, responsibilities, and technology stack for each position.

**Skills**
Technical competencies organized by category:

- Languages
- AI & LLM
- ML & Data
- Backend
- Frontend & Mobile
- DevOps & Tools

**Projects**
Selected portfolio projects featuring:

- Project description and purpose
- Technology stack
- Links to GitHub repository
- Live demo and deployment links where available
